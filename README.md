# Nightdance iOS

Nightdance iOS is a mobile application for Nightdance.

## Getting Started

```bash
git clone https://github.com/nightdance-korea/nightdance-ios.git

cd nightdance-ios

open NightdanceI/Nightdance.xcodeproj
```
